# teamlead-test

## 1. Установить необходимые модули для проекта
```
npm install
```

## 2. Запустить frontend сервер
```
npm run serve
```

## 3. Запустить backend сервер
```
node ./src/server.js
```
Frontend запущен на http://localhost:8080, backend на http://localhost:3000
